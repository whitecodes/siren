# Siren - 塞壬唱片播放器

一个简洁优雅的塞壬唱片音乐播放器，专为《明日方舟》玩家打造。

## 功能特点

- **专辑浏览**：查看所有塞壬唱片专辑及歌曲
- **高品质播放**：支持在线播放所有歌曲
- **下载功能**：可将歌曲下载到本地离线收听
- **播放列表**：管理你的播放队列
- **搜索**：快速查找专辑和歌曲
- **深色/浅色主题**：支持跟随系统或手动切换
- **多语言**：支持中文、英文、日文

## 系统要求

- Android 8.0 (API 26) 及以上版本

## 安装

1. 下载最新的 APK 文件
2. 在手机上安装（可能需要允许"安装未知来源应用"）
3. 打开应用即可使用
